# Avirob
Avirob es un robot educativo que sigue la estela de Zowi y OttoDIY, pero con un espíritu de continuidad que trata de ofrecer a la comunidad los recursos que acompañan a este contexto, tanto en archivos de tipo CAD (modificables con FreeCAD), como STL (para imprimir en 3D), así como el software y esquemas de conexión que capacitan al aficionado a construir y programar su propio robot.
